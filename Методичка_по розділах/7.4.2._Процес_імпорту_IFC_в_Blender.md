### 7.4.2. Процес імпорту IFC в Blender

1. **Встановлення та активація аддону BlenderBIM:**
    * **BlenderBIM** — це потужний безкоштовний аддон для Blender, який значно розширює можливості роботи з IFC-файлами (імпорт, експорт, редагування IFC-даних). Його можна завантажити з офіційного сайту та встановити в Blender через меню **Edit -> Preferences -> Add-ons -> Install**.
    * Після встановлення активуйте аддон.
2. **Імпорт IFC:**
    * **File -> Import -> Industry Foundation Classes (.ifc) (BlenderBIM)**.
    * **Налаштування параметрів імпорту (в панелі імпорту):**
        * **Merge Entities by IFC Class/Type (Об'єднувати об'єкти за класом/типом IFC):** Може бути корисним для групування однотипних елементів.
        * **Import Properties (Імпортувати властивості):** Якщо потрібні атрибутивні дані в Blender.
        * **Coordinate System / Offset (Система координат / Зсув):** Важливо для правильного розміщення моделі, особливо якщо вона має великі географічні координати. Іноді потрібно налаштувати зсув, щоб модель опинилася близько до центру світових координат Blender.
        * **Element Filtering (Фільтрація елементів):** Можливість імпортувати лише певні типи IFC-об'єктів.
3. **Перевірка імпортованої моделі:**
    * Огляньте модель, перевірте масштаби, орієнтацію, наявність всіх необхідних елементів.
