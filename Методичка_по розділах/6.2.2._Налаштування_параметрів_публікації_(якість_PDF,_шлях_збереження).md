#### 6.2.2. Налаштування параметрів публікації (якість PDF, шлях збереження)

У діалоговому вікні **"Publish Options"** (Параметри публікації) або в налаштуваннях обраного **"PDF preset"** (через "PDF Options..." або властивості плотера "DWG To PDF.pc3") можна налаштувати наступні важливі параметри:

1. **Місцезнаходження (Location / Path):**
    * Вкажіть папку, куди будуть збережені вихідні PDF-файли. Рекомендується створити окрему папку для фінальних PDF-документів (наприклад, "ПРОЕКТ/[КОД_ПРОЕКТУ]_[НАЗВА_ПРОЕКТУ]/10_PDF_ВИПУСК/").

2. **Тип файлу (File Type):**
    * **Single-sheet file (Односторінковий файл):** Кожен аркуш буде збережено як окремий PDF-файл.
    * **Multi-sheet file (Багатосторінковий файл):** Всі обрані аркуші будуть об'єднані в один багатосторінковий PDF-файл. **Це зазвичай є кращим варіантом для створення єдиної книги документації.**

3. **Іменування файлів (File Naming) (для односторінкових файлів):**
    * Можна налаштувати, як будуть іменуватися окремі PDF-файли (наприклад, на основі імені Листа, номера Листа або їх комбінації).

4. **Якість векторної графіки (Vector quality):**
    * Визначає роздільну здатність (DPI - dots per inch) для векторної графіки. Вищі значення забезпечують кращу якість (більш плавні лінії), але збільшують розмір файлу. Типові значення: 300, 600, 1200 DPI. Для більшості креслень 600 DPI є хорошим компромісом.

5. **Якість растрової графіки (Raster image quality):**
    * Визначає роздільну здатність для растрових зображень, вставлених у креслення.

6. **Обробка ліній (Line Handling):**
    * **Merge lines (Об'єднувати лінії):** Може зменшити розмір файлу, але іноді призводить до некоректного відображення перетинів ліній. Рекомендується тестувати.
    * **Convert text to geometry (Перетворювати текст на геометрію):** Не рекомендується, оскільки текст стане нередагованим та не буде доступним для пошуку в PDF. Краще вбудовувати шрифти.

7. **Включення інформації (Include information):**
    * **Layer information (Інформація про шари):** Якщо увімкнено, користувач зможе вмикати/вимикати видимість шарів у PDF-файлі. Дуже корисна функція.
    * **Hyperlinks (Гіперпосилання):** Якщо в кресленні є гіперпосилання, вони будуть збережені в PDF.
    * **Bookmarks (Закладки):** Можна автоматично створювати закладки в PDF на основі назв Листів або іменованих видів.
    * **Font handling (Обробка шрифтів):**
        * **Capture fonts used in drawing (Вбудовувати шрифти, використані в кресленні): Дуже рекомендується.** Забезпечує правильне відображення тексту в PDF, навіть якщо на комп'ютері користувача відсутні використані шрифти.
        * **Convert all text to geometry (Перетворювати весь текст на геометрію):** Уникайте цієї опції, якщо можливе вбудовування шрифтів.

8. **Захист PDF (PDF Security Options) (якщо доступно в налаштуваннях плотера):**
    * Можливість встановлення паролів на відкриття, друк, редагування PDF-файлу.

9. **Інші параметри:**
    * **Prompt for name (Запитувати ім'я):** Чи запитувати ім'я файлу перед кожною
