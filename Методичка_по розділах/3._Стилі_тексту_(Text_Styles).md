### 3. Стилі тексту (Text Styles):
* Створені корпоративні стилі тексту для основного тексту, заголовків, розмірних чисел, приміток.
* **Параметри стилю:** Назва стилю, ім'я шрифту (наприклад, "Arial Narrow CYR", "Times New Roman CYR", або спеціалізований креслярський шрифт, що відповідає ДСТУ), висота тексту (часто встановлюється як 0 для можливості задання висоти при створенні тексту, або фіксована висота для анотативних стилів), коефіцієнт стиснення, кут нахилу.
* **Анотативність:** Налаштовані анотативні стилі тексту, які автоматично масштабуються залежно від масштабу видового екрана.
