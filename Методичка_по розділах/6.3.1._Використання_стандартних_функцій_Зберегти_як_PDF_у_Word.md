### 6.3.1. Використання стандартних функцій "Зберегти як PDF" у Word

Сучасні версії Microsoft Word (починаючи з Word 2007 Service Pack 2 та новіші) мають вбудовану функцію збереження документів безпосередньо у формат PDF. Це найпростіший та найдоступніший спосіб конвертації.

**Процес конвертації:**

1. **Відкрийте документ Word (.docx або .doc), який потрібно конвертувати** (наприклад, файл Відомості обсягів робіт).
2. **Перейдіть до меню "Файл" (File).**
3. **Оберіть опцію "Зберегти як" (Save As) або "Експорт" (Export).**
    * **Якщо обрано "Зберегти як":**
        * У діалоговому вікні "Зберегти як" оберіть місце збереження файлу.
        * У полі "Тип файлу" (Save as type) зі списку, що випадає, оберіть **"PDF (*.pdf)"**.
        * Натисніть кнопку **"Параметри..." (Options...)** для доступу до додаткових налаштувань (див. нижче).
        * Натисніть **"Зберегти" (Save)**.
    * **Якщо обрано "Експорт":**
        * Оберіть **"Створити PDF/XPS документ" (Create PDF/XPS Document)**.
        * Натисніть кнопку **"Створити PDF/XPS" (Create PDF/XPS)**.
        * Відкриється діалогове вікно, аналогічне "Зберегти як", де потрібно обрати місце збереження, переконатися, що тип файлу "PDF", та за потреби налаштувати параметри через кнопку "Параметри...".
        * Натисніть **"Опублікувати" (Publish)**.

**Налаштування параметрів PDF (кнопка "Параметри..." / "Options..."):**

У діалоговому вікні "Параметри" можна налаштувати декілька важливих аспектів конвертації:

* **Діапазон сторінок (Page range):**
    * **"Усі" (All):** Конвертувати весь документ (за замовчуванням).
    * **"Поточна сторінка" (Current page):** Конвертувати лише активну сторінку.
    * **"Виділений фрагмент" (Selection):** Конвертувати лише попередньо виділену частину документа (якщо підтримується).
    * **"Сторінки" (Pages):** Вказати конкретні номери сторінок або діапазони для конвертації (наприклад, 1-5, 8, 10-12).
* **Що публікувати (Publish what):**
    * **"Документ" (Document):** Стандартний варіант.
    * **"Документ із виправленнями" (Document showing markup):** Якщо в документі є записані виправлення (Track Changes), вони будуть відображені в PDF. Для

фінальної документації зазвичай виправлення повинні бути прийняті або відхилені, а ця опція не використовується.

*   **Включити недруковані відомості (Include non-printing information):**
    *   **"Властивості документа" (Document properties):** Включає метадані документа (автор, назва, ключові слова) в PDF.
    *   **"Теги структури документа для спеціальних можливостей" (Document structure tags for accessibility):** Покращує доступність PDF для людей з обмеженими можливостями та для програмного аналізу структури документа. Рекомендується вмикати.
*   **Параметри PDF (PDF options):**
    *   **"Сумісність з PDF/A" (PDF/A compliant):** Дуже важлива опція для архівного зберігання. PDF/A – це стандарт ISO, розроблений для довгострокового збереження електронних документів. Він гарантує, що файл буде відкриватися та коректно відображатися в майбутньому. Якщо документація призначена для архіву, цю опцію слід увімкнути. Це може дещо збільшити розмір файлу та накласти певні обмеження (наприклад, всі шрифти повинні бути вбудовані, не можна використовувати прозорість або шифрування).
    *   **"Оптимізувати для (Optimize for):"**
        *   **"Стандартний (публікація в Інтернеті та друк)" (Standard (publishing online and printing)):** Забезпечує хороший баланс між якістю та розміром файлу. Зазвичай підходить для більшості завдань.
        *   **"Мінімальний розмір (публікація в Інтернеті)" (Minimum size (publishing online)):** Створює PDF-файли меншого розміру за рахунок можливого зниження якості зображень. Може бути корисним для передачі електронною поштою, але для фінальної документації краще обирати стандартну якість.
*   **Інструменти -> Стиснути рисунки (Tools -> Compress Pictures) (доступно перед збереженням через діалог "Зберегти як"):**
    *   Дозволяє налаштувати роздільну здатність для растрових зображень у документі, що впливає на якість та розмір PDF. Для друку рекомендується не менше 220 ppi, для веб – 150 ppi або 96 ppi.

**Переваги вбудованої функції:**

*   Простота та доступність (не потребує встановлення додаткового ПЗ).
*   Добре зберігає форматування оригінального документа Word.
*   Підтримка створення PDF/A.
*   Можливість вбудовування шрифтів (зазвичай відбувається автоматично при стандартних налаштуваннях).
