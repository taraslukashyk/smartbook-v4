### 9.2.2. Необхідне програмне забезпечення на планшетах

Для ефективної роботи з цифровою документацією та інформаційними моделями на планшетах повинно бути встановлене відповідне програмне забезпечення.

1. **Переглядачі PDF-документів (PDF Viewers/Readers):**
    * **Призначення:** Для перегляду робочих креслень, відомостей обсягів робіт, пояснювальних записок та іншої текстової документації у форматі PDF.
    * **Основні функції:**
        * Відкриття та перегляд PDF-файлів.
        * Масштабування, панорамування.
        * Пошук по тексту.
        * Навігація по закладках (bookmarks) та мініатюрах сторінок.
        * Підтримка гіперпосилань.
        * Можливість додавання коментарів, нотаток, виділення тексту (якщо потрібна функція рецензування).
    * **Рекомендоване ПЗ:**
        * **Adobe Acrobat Reader (Android, iOS, Windows):** Безкоштовний, стандарт де-факто.
        * **Foxit PDF Reader Mobile (Android, iOS) / Foxit PDF Editor (Windows):** Хороша альтернатива з багатим функціоналом.
        * **Xodo PDF Reader & Editor (Android, iOS, Windows):** Популярний безкоштовний додаток з функціями анотування.
        * **Google PDF Viewer (вбудований в Android).**
        * **Вбудовані переглядачі PDF в операційних системах.**

2. **Переглядачі IFC-моделей (IFC Viewers):**
    * **Призначення:** Для перегляду тривимірних інформаційних моделей у форматі IFC, аналізу геометрії та атрибутивної інформації.
    * **Основні функції:**
        * Відкриття IFC-файлів (різних версій).

- 3D-навігація по моделі (обертання, панорамування, зум).
- Вибір об'єктів та перегляд їхніх властивостей (атрибутів).
- Управління видимістю об'єктів (приховування, ізоляція, прозорість).
- Створення розрізів та перерізів.
- Вимірювання відстаней, кутів (іноді площ).
- Робота зі структурою моделі.
    - **Рекомендоване ПЗ (для мобільних платформ):**
        - **BIMvision Mobile (Android, iOS – перевірити актуальну доступність та функціонал мобільної версії):** Якщо доступна повноцінна мобільна версія, це може бути хорошим варіантом, враховуючи рекомендацію BIMvision для настільних ПК.
        - **BIMx (Graphisoft) (Android, iOS):** Хоча розроблений для моделей ArchiCAD, BIMx може відкривати IFC-файли. Має хороший інтерфейс та функцію "Hyper-model" для зв'язку 3D-моделі з 2D-кресленнями.
        - **Trimble Connect for Mobile (Android, iOS):** Частина хмарної платформи Trimble Connect, дозволяє переглядати IFC та інші формати моделей.
        - **Autodesk BIM 360 / Autodesk Construction Cloud (ACC) Mobile App (Android, iOS):** Якщо компанія використовує ці хмарні платформи, мобільні додатки надають доступ до моделей та документів, включаючи перегляд IFC.
        - **Dalux Field (Android, iOS):** Спеціалізований додаток для будівельного майданчика з підтримкою IFC.
        - **Fieldwire (Android, iOS):** Платформа для управління будівельними проєктами, включає переглядач креслень та моделей.
        - **Solibri Anywhere Mobile (перевірити доступність):** Якщо є мобільна версія.
        - **EveBIM (Abvent) (можливо, є мобільні рішення).**
    - **Важливо:** Функціонал мобільних IFC-переглядачів може бути дещо обмеженим порівняно з їхніми настільними версіями, особливо щодо обробки дуже великих моделей або складного аналізу. Необхідно тестувати продуктивність на обраних планшетах з типовими для компанії IFC-файлами.

3. **Хмарні сервіси та платформи для спільної роботи (CDE – Common Data Environment):**
    - Якщо компанія використовує CDE (наприклад, Autodesk Construction Cloud, Trimble Connect, Asite, Viewpoint, BIMserver.center), відповідні мобільні додатки повинні бути встановлені для доступу до актуальної документації, моделей, завдань, RFI.

4. **Інші корисні додатки:**
    - **Камера та галерея:** Для фотофіксації.
    - **Додатки для створення нотаток та ескізів:** (наприклад, Microsoft OneNote, Evernote, Google Keep, спеціалізовані додатки для малювання зі стилусом).
    - **Офісні пакети (для перегляду DOCX, XLSX):** (наприклад, Microsoft Office Mobile, Google Docs/Sheets).
    - **Файлові менеджери:** Для управління файлами на пристрої.
    - **Додатки для відеозв'язку:** (наприклад, Microsoft Teams, Zoom, Google Meet) для комунікації з офісом.

**Забезпечення безпеки даних на мобільних пристроях:**

*   Використання паролів, PIN-кодів або біометричної автентифікації для доступу до планшета.
*   Шифрування даних на пристрої.
*   Використання корпоративних рішень для управління мобільними пристроями (MDM – Mobile Device Management), якщо це необхідно.
*   Регулярне оновлення операційної системи та додатків.

Правильний вибір обладнання та програмного забезпечення є важливою передумовою для успішного впровадження цифрових інструментів на будівельному майданчику.
