## 2. Шаблони для програм Credo або Robur (для проєктування доріг, генплану):

Програми Credo та Robur мають власні формати шаблонів проєктів або налаштувань. Корпоративні шаблони для них можуть включати:

*   **Налаштування проєкту:**
    *   Одиниці виміру, система координат.
    *   Класифікатори шарів, точок, ліній, об'єктів.
*   **Стилі відображення:**
    *   Налаштування для відображення елементів на плані, поздовжніх та поперечних профілях.
    *   Стилі умовних позначень.
*   **Параметри розрахунків:**
    *   Стандартні параметри для розрахунку земляних мас, конструкцій дорожнього одягу.
*   **Шаблони відомостей та креслень:**
    *   Налаштовані форми для генерації відомостей (наприклад, відомість елементів плану та профілю, відомість земляних робіт).
    *   Шаблони оформлення аркушів креслень.
*   **Бібліотеки типових конструкцій:**
    *   Набори типових поперечників, елементів облаштування дороги.
