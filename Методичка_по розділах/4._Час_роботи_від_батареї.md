## 4. Час роботи від батареї:
*   Планшет повинен забезпечувати роботу протягом повного робочого дня (8-10 годин) без підзарядки.
*   Ємність батареї (в мАг) є важливим показником, але реальний час роботи залежить від багатьох факторів (яскравість екрану, інтенсивність використання, тип завдань).
*   **Можливість гарячої заміни батареї (Hot-Swappable Battery):** Деякі захищені планшети дозволяють замінювати батарею без вимкнення пристрою, що дуже зручно для тривалої автономної роботи.

5.  **Операційна система (ОС):**
    *   **Android:** Найбільш поширений варіант для захищених планшетів. Велика кількість доступних додатків, включаючи IFC-переглядачі та PDF-рідери. Важливо звертати увагу на версію Android та можливість її оновлення.
    *   **Windows:** Планшети на Windows (зазвичай Windows 10/11 Pro) можуть бути зручними, якщо потрібна сумісність з настільними Windows-додатками або специфічним корпоративним ПЗ. Вони часто більш продуктивні, але можуть бути дорожчими та мати менший час автономної роботи.
    *   **iOS (iPad):** iPad-и відомі своєю продуктивністю та якісними екранами, але вони менш захищені (потребують спеціальних захисних чохлів) і мають вищу ціну. Екосистема додатків для BIM також добре розвинена.

6.  **Комунікаційні можливості:**
    *   **Wi-Fi:** Обов'язково, з підтримкою сучасних стандартів (наприклад, Wi-Fi 5/802.11ac або Wi-Fi 6/802.11ax) для швидкого зв'язку.
    *   **Мобільний зв'язок (4G LTE / 5G):** Дуже бажано для доступу до даних та синхронізації в місцях, де немає Wi-Fi. Потребує SIM-карти.
    *   **Bluetooth:** Для підключення бездротових аксесуарів (навушники, клавіатури, GPS-приймачі).
    *   **GPS/GLONASS/Galileo:** Вбудований модуль геолокації для визначення місцезнаходження на об'єкті, що може використовуватися деякими додатками для прив'язки фотографій або нотаток до координат.

7.  **Порти та роз'єми:**
    *   USB-C (бажано з підтримкою швидкої зарядки та передачі даних).
    *   Слот для карт пам'яті microSD.
    *   Аудіороз'єм (3.5 мм).
    *   Можливо, спеціалізовані порти для підключення додаткового обладнання (сканери штрих-кодів, RFID-рідери — залежно від потреб).

8.  **Камери:**
    *   **Основна камера (тильна):** З достатньою роздільною здатністю (наприклад, 8-13 Мп) та автофокусом для фотофіксації виконаних робіт, дефектів, зчитування QR-кодів. Наявність спалаху.
    *   **Фронтальна камера:** Для відеодзвінків.

9.  **Додаткові функції та аксесуари:**
    *   **Сканер відбитків пальців або розпізнавання обличчя:** Для безпечного доступу.
    *   **Програмовані кнопки:** Можуть бути налаштовані для швидкого запуску часто використовуваних додатків або функцій.
    *   **Наявність та доступність аксесуарів:** Захисні чохли, плівки на екран, стилуси, док-станції, автомобільні кріплення, ремені для перенесення.

10. **Вартість та гарантія:**
