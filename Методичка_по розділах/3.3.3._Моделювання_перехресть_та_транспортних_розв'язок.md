### 3.3.3. Моделювання перехресть та транспортних розв'язок

Infraworks надає зручні інструменти для концептуального моделювання перехресть автомобільних доріг, як в одному, так і в різних рівнях.

* **Перехрестя в одному рівні (At-Grade Intersections):**
    * При перетині двох або більше компонентних доріг Infraworks автоматично створює перехрестя.

- Користувач може налаштувати параметри перехрестя:
    - **Радіуси заокруглень (Curb Returns):** Задання радіусів для плавного сполучення доріг.
    - **Смуги руху:** Додавання додаткових смуг для повороту, зміна ширини смуг.
    - **Острівці безпеки та напрямні острівці:** Створення та редагування їх геометрії.
    - **Світлофорне регулювання:** Можливість додати (концептуально) світлофори.
    - **Розмітка:** Автоматичне або ручне нанесення дорожньої розмітки.

- **Кільцеві розв'язки (Roundabouts):**
    - Infraworks має спеціалізований інструмент для створення кільцевих розв'язок.
    - Задаються основні параметри кільця (діаметр центрального острівця, ширина проїзної частини кільця, кількість смуг) та параметри примикань.
    - Можливість аналізу геометрії кільця на відповідність стандартам (наприклад, за допомогою вбудованих правил або шляхом експорту в Civil 3D для більш детального аналізу).

- **Транспортні розв'язки в різних рівнях (Interchanges/Grade Separations):**
    - Infraworks дозволяє концептуально проєктувати з'їзди та шляхопроводи для створення розв'язок.
    - Моделювання шляхопроводів виконується аналогічно до моделювання мостів.
    - З'їзди (рампи) створюються як окремі компонентні дороги, що сполучають основні напрямки.
    - Процес вимагає більш ретельного планування та роботи з вертикальною геометрією для забезпечення необхідних габаритів та ухилів.

- **Візуалізація та аналіз:**
    - 3D-модель перехрестя або розв'язки дозволяє візуально оцінити її геометрію, оглядовість, зручність проїзду.
    - Можливість базової симуляції транспортних потоків для оцінки ефективності проєктного рішення на ранніх етапах.

Інструменти Infraworks для моделювання перехресть та розв'язок спрямовані на швидке створення варіантів та їх візуальну оцінку, що є дуже корисним на стадії обґрунтування інвестицій або розробки передпроєктних пропозицій.
