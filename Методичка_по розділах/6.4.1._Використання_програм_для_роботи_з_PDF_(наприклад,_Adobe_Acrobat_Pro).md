### 6.4.1. Використання програм для роботи з PDF (наприклад, Adobe Acrobat Pro)

Для об'єднання PDF-файлів потрібне спеціалізоване програмне забезпечення. Безкоштовні програми для читання PDF (наприклад, Adobe Acrobat Reader) зазвичай не мають такої функції.

* **Adobe Acrobat Pro:**
    * Є найбільш потужним та функціональним інструментом для роботи з PDF, включаючи об'єднання файлів.
    * **Способи об'єднання:**
        * **"Об'єднати файли в PDF" (Combine Files into PDF):** File -> Create -> Combine Files into a Single PDF. Дозволяє додати декілька PDF-файлів (а також файлів інших форматів, які Acrobat може конвертувати), впорядкувати їх у потрібній послідовності та об'єднати в один PDF.
        * **Перетягування сторінок (Drag and Drop):** Можна відкрити декілька PDF-

*   ■ **Вставка сторінок (Insert Pages):** В інструменті "Організувати сторінки" можна вставити сторінки з іншого PDF-файлу в поточний документ у вказане місце.
*   ● **Інші програми для роботи з PDF:**
    *   ○ **Foxit PDF Editor Pro, Nitro PDF Pro, ABBYY FineReader PDF:** Платні альтернативи Adobe Acrobat Pro, які також мають функції для об'єднання PDF-файлів.
    *   ○ **Онлайн-сервіси для об'єднання PDF:** Існує багато безкоштовних веб-сервісів (наприклад, Smallpdf, iLovePDF, PDF2Go), які дозволяють завантажувати та об'єднувати PDF-файли онлайн. Однак, при використанні онлайн-сервісів слід бути **дуже обережними з конфіденційною проєктною документацією**, оскільки файли завантажуються на сторонні сервери. Для офіційних документів компанії рекомендується використовувати настільне ПЗ.
    *   ○ **Безкоштовне ПЗ:** Деякі безкоштовні програми (наприклад, PDFsam Basic – PDF Split and Merge) спеціалізуються на операціях розділення та об'єднання PDF-файлів.
