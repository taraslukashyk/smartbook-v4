# РОЗДІЛ 3. СТВОРЕННЯ ІНФОРМАЦІЙНОЇ МОДЕЛІ ТРАНСПОРТНОЇ СПОРУДИ

Центральним елементом BIM-процесу є **інформаційна модель (ІМ)**. Цей розділ присвячений аспектам створення моделі автомобільних доріг та інфраструктурних об'єктів.

### Зміст розділу:
- **[3.1. Вибір Програмного Забезпечення](./01-software-choice.md)** — порівняння Civil 3D, Credo та Robur.
- **[3.2.1-3.2.2. План та Профіль Траси](./02-modeling-process.md)** — імпорт даних та геометрія осі.
- **[3.2.3. Конструктивні Елементи](./03-constructive-elements.md)** — земляне полотно, одяг, мережі.
- **[3.2.4. Атрибутивні Дані](./04-attribute-data.md)** — насичення моделі інформацією (Property Sets).
- **[3.2.5. Контроль Якості та Колізії](./05-quality-control.md)** — перевірка моделі в Navisworks та Solibri.
- **[3.3. Використання Infraworks](./06-infraworks.md)** — концептуальне моделювання та контекст.
- **[3.4. Моделювання Штучних Споруд](./07-structures.md)** — мости, тунелі та розрахунок конструкцій.

Використовуйте бокове меню для навігації між підрозділами.
