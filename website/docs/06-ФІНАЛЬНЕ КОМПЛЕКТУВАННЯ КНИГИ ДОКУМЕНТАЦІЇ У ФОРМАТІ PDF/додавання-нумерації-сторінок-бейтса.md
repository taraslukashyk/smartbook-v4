## Додавання Нумерації Сторінок (Бейтса)

Після того, як всі окремі PDF-файли (креслення, відомості, текстові документи) об'єднані в єдиний документ або том, наступним важливим кроком є додавання наскрізної нумерації сторінок. Це забезпечує унікальну ідентифікацію кожної сторінки в межах комплекту документації, полегшує посилання на конкретні сторінки та є вимогою для багатьох офіційних процедур (наприклад, подання на експертизу, архівування).

Термін **"Нумерація Бейтса" (Bates Numbering)** походить з юридичної практики, де він використовується для послідовної нумерації сторінок у великих обсягах документів для судових справ. У контексті проєктної документації це, по суті, означає додавання унікального ідентифікатора (зазвичай номера) до кожної сторінки об'єднаного PDF-файлу.

### 6.5.1. Використання функції "Bates Numbering" або аналогічних інструментів у програмах для роботи з PDF

Більшість професійних програм для роботи з PDF, таких як **Adobe Acrobat Pro, Foxit PDF Editor Pro, Nitro PDF Pro**, мають вбудовані інструменти для додавання нумерації Бейтса або аналогічної функції колонтитулів.

**Процес додавання нумерації на прикладі Adobe Acrobat Pro:**

1. **Відкрийте об'єднаний PDF-файл**, до якого потрібно додати нумерацію.
2. **Перейдіть до інструментів редагування:**
    * Зазвичай це можна знайти в меню **"Інструменти" (Tools)** -> **"Редагувати PDF" (Edit PDF)** -> **"Колонтитули" (Header & Footer)** -> **"Додати" (Add)**.
    * Або через **"Інструменти" (Tools)** -> **"Штампи" (Stamps)** -> **"Нумерація Бейтса" (Bates Numbering)** -> **"Додати" (Add)**. (Інтерфейс може дещо відрізнятися залежно від версії Acrobat).
3. **Налаштування параметрів нумерації (у діалоговому вікні "Додати колонтитули" або "Нумерація Бейтса"):**
    * **Розміщення (Placement):**
        * **Поля колонтитулів:** Зазвичай нумерація розміщується в нижньому або верхньому колонтитулі. Acrobat пропонує шість основних позицій: Верхній лівий, Верхній центральний, Верхній правий, Нижній лівий, Нижній центральний, Нижній правий.
        * **Відступи від країв (Margins):** Можна точно налаштувати відступи від країв аркуша для розміщення номера.
    * **Формат номера (Number Format):**
        * **Кількість цифр (Number of digits):** Визначає, скільки цифр буде в номері сторінки (наприклад, 3 цифри дадуть номери 001, 002, ... 010, ... 100). Це

*   **Початковий номер (Start page number):** Задає номер, з якого почнеться нумерація (зазвичай 1).
*   **Префікс (Prefix):** Текст або символи, що додаються перед номером сторінки (наприклад, "Арк.", "Стор.", код проєкту, номер тому).
*   **Суфікс (Suffix):** Текст або символи, що додаються після номера сторінки.
*   **Шрифт та вигляд (Font and Appearance):**
    *   **Шрифт (Font):** Оберіть шрифт для номерів сторінок (наприклад, Arial, Times New Roman).
    *   **Розмір шрифту (Size):** Встановіть розмір шрифту (наприклад, 8, 9, 10 пунктів). Номер не повинен бути надто великим, щоб не заважати основному вмісту, але достатньо читабельним.
    *   **Колір шрифту (Color):** Зазвичай чорний.
    *   **Підкреслення (Underline)** та інші ефекти (якщо потрібні).
*   **Діапазон сторінок (Page Range Options):**
    *   Можна застосувати нумерацію до всіх сторінок або до обраного діапазону.
    *   Можливість застосовувати різні налаштування нумерації для різних діапазонів сторінок (наприклад, не нумерувати титульний аркуш або обкладинку).

4.  **Попередній перегляд (Preview):**
    *   Більшість програм дозволяють переглянути, як буде виглядати нумерація на сторінці, перед її застосуванням.
5.  **Застосування нумерації:**
    *   Натисніть "ОК" або "Застосувати" (Apply). Нумерація буде додана до всіх обраних сторінок документа.
6.  **Збереження файлу:**
    *   Після додавання нумерації обов'язково збережіть PDF-файл (рекомендується зберегти під новим ім'ям або як нову версію, щоб мати можливість повернутися до варіанту без наскрізної нумерації, якщо це буде потрібно).

### 6.5.2. Налаштування формату нумерації згідно з вимогами проєкту або внутрішніми стандартами

Формат наскрізної нумерації сторінок повинен відповідати загальним вимогам до оформлення проєктної документації та може бути деталізований у:

*   **Вимогах Замовника:** Іноді замовник може висувати специфічні вимоги до формату нумерації.
*   **Вимогах експертних організацій:** Для подання на експертизу може бути потрібен певний формат.
*   **Внутрішніх корпоративних стандартах [Назва Вашої Організації/Компанії]:** Компанія може мати власний стандарт щодо формату та розміщення наскрізної нумерації для всієї вихідної документації.

**Типові елементи формату нумерації:**

*   **Простий номер сторінки:** Наприклад, "1", "2", "15", "125".

* **Номер сторінки з префіксом:**
    * "Арк. 1", "Арк. 125"
    * "Стор. 1", "Стор. 125"
    * "[КодПроєкту]-1", "[КодПроєкту]-125"
    * "[НомерТому]-1", "[НомерТому]-125" (якщо документація розбита на томи, і кожен том має свою наскрізну нумерацію).
* **Номер сторінки з фіксованою кількістю цифр:** Наприклад, при 3 цифрах: "001", "015", "125".
* **Комбінований формат:** Наприклад, "[Префікс] - [Номер сторінки] з [Загальна кількість сторінок у документі]". Деякі програми дозволяють автоматично вставляти загальну кількість сторінок.

### 6.5.3. Розміщення нумерації у колонтитулах

Найбільш поширеним місцем для розміщення наскрізної нумерації сторінок є:

* **Нижній колонтитул (Footer):**
    * **По центру (Bottom Center):** Найчастіше використовуваний варіант.
    * **Праворуч (Bottom Right):** Також поширений.
    * **Ліворуч (Bottom Left).**
* **Верхній колонтитул (Header):**
    * Використовується рідше для основної нумерації, але може містити іншу інформацію (назву розділу, дату). Якщо нумерація розміщується вгорі, то також по центру, праворуч або ліворуч.

**Рекомендації щодо розміщення:**

* **Узгодженість:** Нумерація повинна бути розміщена в одному й тому ж місці на всіх сторінках документа (за винятком, можливо, титульних аркушів або спеціальних сторінок, де нумерація може бути прихована).
* **Відступи:** Забезпечте достатні відступи від країв аркуша та від основного тексту, щоб номер не "зливався" з іншим вмістом і не обрізався при друці.
* **Читабельність:** Шрифт та розмір повинні бути такими, щоб номер легко читався, але не відволікав від основного змісту креслення або тексту.
* **Не перекривати важливу інформацію:** Переконайтеся, що нумерація не накладається на основний напис (штамп), рамку креслення або іншу важливу інформацію.

Після додавання наскрізної нумерації необхідно ретельно перевірити весь документ, щоб переконатися, що нумерація застосована коректно до всіх сторінок, має правильний формат та не створює проблем з читабельністю.
