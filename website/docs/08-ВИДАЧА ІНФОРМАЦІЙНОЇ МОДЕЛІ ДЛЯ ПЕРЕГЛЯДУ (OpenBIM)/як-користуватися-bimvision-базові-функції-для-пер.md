## Як користуватися BIMvision: базові функції для перегляду моделі та інформації

Розглянемо основні кроки та функції при роботі з IFC-моделлю в BIMvision.

**Завантаження та встановлення BIMvision:**

*   Завантажте інсталяційний файл з офіційного сайту BIMvision (bimvision.eu).
*   Встановіть програму, дотримуючись інструкцій.

**Відкриття IFC-файлу:**

*   Запустіть **BIMvision**.
*   **File -> Open** (Файл -> Відкрити) або натисніть на іконку папки.
*   Оберіть ваш **.ifc** або **.ifczip** файл.
*   Модель завантажиться у головне вікно програми.

**Обертання (Orbit): Затисніть середню кнопку миші (коліщатко) та рухайте мишею.**

**Панорамування (Pan): Затисніть клавішу Shift + середню кнопку миші та рухайте мишею.** Або використовуйте праву кнопку миші (залежно від налаштувань).

**Зумування (Zoom): Прокручуйте коліщатко миші вперед/назад.**

**"Прогулянка" (Walk/Fly): Деякі режими навігації дозволяють переміщатися по моделі, як у відеогрі (часто за допомогою клавіш W, A, S, D).**

**Fit to View (Вписати у вигляд): Кнопка для відображення всієї моделі у вікні.**

### Вибір об'єктів:

**Клацніть лівою кнопкою миші на будь-якому елементі моделі, щоб його вибрати. Обраний об'єкт зазвичай підсвічується.**

**Можна вибирати декілька об'єктів, утримуючи клавішу Ctrl або Shift.**

### Перегляд властивостей (атрибутів) об'єкта:

**Після вибору об'єкта, у правій частині інтерфейсу (або в окремому вікні "Properties") відобразиться список його властивостей, згрупованих за наборами (Property Sets).**

**Прокрутіть список, щоб побачити всі доступні атрибути (назва, тип, матеріал, розміри, користувацькі властивості).**

### Вимірювання:

**На панелі інструментів знайдіть інструменти для вимірювання (зазвичай іконка лінійки).**

**Point-to-Point Distance (Відстань точка-точка): Клацніть на двох точках для вимірювання відстані між ними.**

**Edge Distance (Довжина ребра): Клацніть на ребрі об'єкта.**

**Результати вимірювань відображаються на екрані або в окремому вікні.**

### Створення розрізів (Sectioning):

**Знайдіть інструменти для створення розрізів (зазвичай іконка з площиною, що перерізає куб).**

**Увімкніть одну або декілька площин розрізу (X, Y, Z).**

**Переміщуйте площину розрізу за допомогою маніпуляторів або вводячи значення зсуву, щоб побачити внутрішню структуру моделі.**

**Можна змінювати напрямок розрізу.**

### Управління видимістю та структурою моделі:

**IFC Structure Tree (Дерево структури IFC): Зазвичай доступне в окремій панелі. Відображає ієрархію: Проєкт -> Сайт -> Будівля/Споруда -> Поверхи/Ділянки -> Елементи.**

**Клацаючи на вузлах дерева, можна вибирати групи об'єктів.**

**Через контекстне меню на об'єктах або вузлах дерева можна:**

Hide (Приховати) / Show (Показати).

Isolate (Ізолювати / Показати тільки обране).

Set Transparency (Встановити прозорість).

Change Color (Змінити колір) (тимчасово, для візуального аналізу).

Збереження точок зору (Viewpoints):

Налаштуйте потрібний ракурс, видимість об'єктів, розрізи.

**Збережіть поточний вигляд як іменовану точку зору для швидкого доступу в майбутньому.**

**Освоєння цих базових функцій BIMvision (або аналогічного IFC-переглядача) дозволить ефективно працювати з інформаційною моделлю, отримувати необхідні дані та аналізувати проєктні рішення.**
