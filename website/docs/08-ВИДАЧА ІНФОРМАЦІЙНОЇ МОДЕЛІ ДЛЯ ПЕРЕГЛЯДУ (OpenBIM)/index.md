# ВИДАЧА ІНФОРМАЦІЙНОЇ МОДЕЛІ ДЛЯ ПЕРЕГЛЯДУ (OpenBIM)

Завершення етапу проєктування та створення детальної інформаційної моделі (ІМ) транспортної споруди не є кінцевою точкою її життєвого циклу. Навпаки, це лише початок її активного використання на наступних етапах — будівництва, експлуатації, а в майбутньому, можливо, реконструкції чи демонтажу. Для забезпечення ефективної передачі та використання накопиченої в моделі інформації всіма учасниками процесу, незалежно від програмного забезпечення, яке вони використовують, ключову роль відіграють принципи OpenBIM та відкриті формати даних, насамперед IFC (Industry Foundation Classes).

Цей розділ методички присвячений питанням підготовки та видачі інформаційної моделі транспортної споруди у форматі, придатному для перегляду, аналізу та використання третіми сторонами. Розглядаються призначення та переваги такого підходу, особливості формату IFC, процес експорту моделі, рекомендоване програмне забезпечення для перегляду IFC-моделей, а також вимоги до насичення моделі інформацією та контролю її якості перед видачею.
