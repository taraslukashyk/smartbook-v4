## Використання Програм для Перегляду IFC

Після того, як інформаційна модель експортована у формат IFC, для її перегляду, аналізу та використання третіми сторонами (будівельниками, експлуатуючими організаціями, замовником), які можуть не мати доступу до основного BIM-ПЗ, використовуються спеціалізовані програми — IFC-переглядачі (IFC Viewers). Існує багато таких програм, як безкоштовних, так і комерційних.

### 8.4.1. Рекомендоване ПЗ: BIMvision (опис можливостей)

**BIMvision** — це безкоштовний (з можливістю розширення функціоналу за допомогою платних плагінів) IFC-переглядач, розроблений компанією **Datacomp (Польща)**. Він здобув популярність завдяки своїй швидкості, простоті використання та досить широкому функціоналу для перегляду та базового аналізу IFC-моделей.

Основні можливості BIMvision (безкоштовна версія):

**Перегляд 3D-моделі:**

*   **Відкриття IFC-файлів: Підтримка різних версій IFC (включаючи IFC2x3, IFC4).**
*   **Навігація по моделі: Інтуїтивні інструменти для обертання (orbit), панорамування (pan), зумування (zoom) моделі. Різні режими навігації (наприклад, "прогулянка").**
*   **Стандартні види: Швидкий перехід до стандартних видів (зверху, знизу, спереду, ззаду, зліва, справа, ізометрія).**
*   **Режими відображення: Можливість зміни режиму відображення моделі (наприклад, каркасний, з затіненням, з ребрами).**

**Вибір об'єктів:** Можливість вибору окремих елементів моделі кліком миші.

**Відображення властивостей (Properties):** При виборі об'єкта у спеціальному вікні відображаються всі його атрибути (**Property Sets**), що були експортовані в IFC-файл (наприклад, ім'я, тип, матеріал, розміри, користувацькі властивості).

**Пошук та фільтрація об'єктів:** Можливість пошуку об'єктів за назвою або значенням атрибута.

**Вимірювання (Measurement):**

*   **Вимірювання відстаней:** Між двома точками, по ребру, мінімальна відстань між об'єктами.
*   **Вимірювання кутів.**
*   **Вимірювання площ** (зазвичай через плагіни).
*   **Відображення координат точок.**

**Створення розрізів та перерізів (Sectioning):**

*   **Динамічні площини розрізу:** Можливість створення до шести динамічних площин розрізу (по осях X, Y, Z), які можна переміщувати та обертати для перегляду внутрішньої структури моделі.
*   **Збереження видів з розрізами.**

**Управління видимістю об'єктів:**

*   **Приховування/Показ об'єктів (Hide/Show):** Можливість тимчасово приховати обрані об'єкти або показати лише обрані.
*   **Прозорість (Transparency):** Зміна рівня прозорості для обраних об'єктів.

**Робота зі структурою моделі (IFC Structure Tree):** Навігатор, що відображає ієрархічну структуру **IFC-моделі** (Project, Site, Building, Storey, Elements). Дозволяє вибирати та керувати видимістю цілих груп об'єктів.

**Збереження точок зору (Viewpoints):** Можливість збереження налаштованих ракурсів камери для швидкого повернення до них.

**Базові функції друку та експорту зображень.**

**Плагіни (Plugins):** BIMvision має модульну структуру. Багато розширених функцій доступні через платні або безкоштовні плагіни, наприклад:

*   **Advanced Reports:** Для створення користувацьких звітів на основі даних моделі.
*   **Clash Detection:** Для виявлення колізій (зазвичай платний).
*   **IFC Export/Split/Merge:** Для маніпуляцій з IFC-файлами.
*   **Gallery:** Для створення зображень.

*   **Швидкість роботи: Добре справляється з великими IFC-моделями.**

### **8.4.2. Інші можливі безкоштовні або комерційні переглядачі IFC**

**Окрім BIMvision, існує низка інших програм для перегляду IFC-файлів:**

A. Безкоштовні IFC-переглядачі:

Solibri Anywhere (раніше Solibri Model Viewer):

*   **Безкоштовна версія потужного інструменту Solibri Model Checker.**
*   **Дозволяє відкривати IFC-файли та моделі Solibri (SMC).**
*   **Хороші можливості для навігації, перегляду властивостей, створення розрізів.**
*   **Не дозволяє виконувати перевірки (це функція платної версії).**

FZKViewer (Karlsruhe Institute of Technology):

*   **Розроблений в німецькому технологічному інституті.**
*   **Підтримує різні версії IFC, має непоганий функціонал для перегляду та аналізу.**

Open IFC Viewer (Open Design Alliance):

*   **Базується на бібліотеках Open Design Alliance.**

Онлайн IFC-переглядачі:

**Деякі веб-платформи та сервіси (наприклад, Autodesk Viewer, BIMcollab ZOOM WebViewer, Trimble Connect for Web) дозволяють завантажувати та переглядати IFC-моделі безпосередньо у веб-браузері без встановлення ПЗ.**

Зручно для швидкого доступу та спільного перегляду, але може мати обмеження щодо розміру файлів та функціоналу порівняно з настільними програмами. Знову ж таки, обережно з конфіденційними даними при використанні публічних онлайн-сервісів.

**Blender з аддоном BlenderBIM:**

Хоча **Blender** є повноцінним 3D-редактором, з аддоном **BlenderBIM** він стає дуже потужним інструментом для роботи з IFC, включаючи детальний перегляд геометрії та всіх атрибутів. Може бути складнішим для непідготовленого користувача порівняно зі спеціалізованими переглядачами.

### B. Комерційні IFC-переглядачі та програми для координації (з функцією перегляду):

**Autodesk Navisworks (Freedom, Simulate, Manage):**

*   **Navisworks Freedom:** Безкоштовний переглядач для файлів NWD та 3D DWF. Може відкривати IFC-файли, якщо вони попередньо конвертовані в NWD за допомогою Navisworks Manage або Simulate.
*   **Navisworks Simulate/Manage:** Потужні комерційні продукти для координації моделей, виявлення колізій, 4D/5D-моделювання. Мають розширені можливості для перегляду та аналізу IFC-моделей.

**Solibri Model Checker:** Комерційний продукт, лідер у галузі перевірки BIM-моделей на відповідність правилам, стандартам та виявлення колізій. Також є чудовим IFC-переглядачем.

**Trimble Connect:** Хмарна платформа для спільної роботи з BIM-моделями, включає потужні інструменти для перегляду IFC.

**BIMcollab ZOOM:** Програма для перегляду, валідації та управління BIM-проблемами, добре працює з IFC та BCF (BIM Collaboration Format).

Вибір конкретного IFC-переглядача залежить від потреб користувача, його технічних навичок та наявності ліцензій. Для простого перегляду та ознайомлення з моделлю зазвичай достатньо безкоштовних програм типу BIMvision або Solibri Anywhere.
