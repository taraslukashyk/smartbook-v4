## Розмір та якість екрану:

*   **Розмір діагоналі:** Зазвичай для роботи з кресленнями та моделями оптимальними є планшети з діагоналлю екрану від 8 до 12 дюймів. Менші екрани можуть бути незручними для перегляду деталей, більші — менш мобільними.
*   **Роздільна здатність:** Full HD (1920x1080 або 1920x1200) або вище для чіткого відображення дрібних деталей на кресленнях.
*   **Яскравість та читабельність на сонці (Sunlight Readability):** Дуже важливий параметр для роботи на відкритому повітрі. Яскравість екрану повинна бути не менше 500-700 ніт, а краще 1000 ніт або більше. Наявність антивідблискового покриття.
*   **Сенсорний екран:**
    *   **Підтримка роботи в рукавичках (Glove Touch):** Багато будівельників працюють в рукавичках.
    *   **Підтримка роботи мокрими пальцями (Wet Touch):** Для використання під дощем.
    *   **Підтримка стилуса (Pen/Stylus Input):** Для більш точного введення, створення нотаток, малювання ескізів. Деякі захищені планшети комплектуються спеціальними стилусами.
