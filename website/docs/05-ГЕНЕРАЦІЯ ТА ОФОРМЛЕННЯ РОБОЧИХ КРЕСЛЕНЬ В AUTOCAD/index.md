# ГЕНЕРАЦІЯ ТА ОФОРМЛЕННЯ РОБОЧИХ КРЕСЛЕНЬ В AUTOCAD

Незважаючи на активний розвиток технологій інформаційного моделювання (BIM) та перехід до 3D-проєктування, традиційні 2D-креслення залишаються важливою та часто обов'язковою частиною проєктної документації, особливо на стадії робочої документації (РД). Вони необхідні для будівельників на майданчику, для проходження експертизи, для архівування та для багатьох інших цілей. Autodesk AutoCAD, завдяки своїй універсальності, гнучкості та широкому розповсюдженню, продовжує бути одним з основних інструментів для створення та оформлення робочих креслень інфраструктурних об'єктів.

Цей розділ методички присвячений принципам та практичним аспектам генерації та оформлення робочих креслень транспортних споруд в середовищі AutoCAD, з урахуванням вимог чинних стандартів України та корпоративних стандартів [Назва Вашої Організації/Компанії]. Розглядаються питання використання шаблонів, роботи з даними, імпортованими з моделюючого ПЗ, оформлення в просторі листа, використання підшивок та контролю якості фінальної документації.
